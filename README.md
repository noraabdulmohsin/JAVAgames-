# 🎯 Guess The Number - Java Game

## 📌 Overview

This is a simple console-based game built in Java where the player tries to guess a randomly generated number within a limited number of attempts.

The project focuses on practicing core programming concepts such as:

* Object-Oriented Programming (OOP)
* Loops and conditionals
* User input handling
* Basic game logic design

---

## 🎮 How It Works

* The game generates a random number within a specific range.
* The player enters guesses through the console.
* After each guess, the game provides feedback:

  * Too high
  * Too low
  * Correct
* The player wins if they guess the number within the allowed attempts.

---

## ⚙️ Features

* Difficulty levels (Easy / Hard)
* Attempt tracking
* Win / Lose conditions
* Clean class-based structure

---

## 🧱 Project Structure

The project is organized using multiple classes:

* `Main` → starts the program
* `Game` → handles game logic
* `Player` → stores player data
* `NumberGenerator` → generates the random number

---

## 🚀 How to Run

1. Clone the repository
2. Open the project in your Java IDE
3. Run the `Main` class
